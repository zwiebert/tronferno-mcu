
#### GPIO Pins ESP32 WLAN-only

  * RF receiver: GPIO-17
  * RF transmitter: GPIO-22
  * Set-Endposition Button: GPIO-6

#### GPIO Pins ESP32 with Ethernet 

  * RF receiver: GPIO-15
  * RF transmitter: GPIO-16
  * Set-Endposition Button: GPIO-6

#### GPIO Pins ESP8266 

  * RF receiver: GPIO-5
  * RF transmitter: GPIO-4
  * Set-Endposition Button: GPIO-0
