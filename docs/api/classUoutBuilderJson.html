let a snprintf like function write JSON array elements directly into our buffer and add a comma

Parameters
fthis function should behave exactly like snprintf(3). We do a second pass based on its return value if buffer was too small.
arr_llennumber of nested arrays arr[arr_llen][]
arr_rlennumber of array elements arr[][arr_rlen]
required_sizeif you know the size of the produced JSON per element, feel free to provide it here. It saves the second pass. Making it too large is wasteful. To increase chunk size use get_a_buffer(), because this parameter will not work for that
Returns
success
