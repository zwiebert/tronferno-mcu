69 /**
70  * \brief Encode a buffer into base64 format
71  *
72  * \param dst destination buffer
73  * \param dlen size of the destination buffer
74  * \param olen number of bytes written
75  * \param src source buffer
76  * \param slen amount of data to be encoded
77  *
78  * \return 0 if successful, or MBEDTLS_ERR_BASE64_BUFFER_TOO_SMALL.
79  * *olen is always updated to reflect the amount
80  * of data that has (or would have) been written.
81  * If that length cannot be represented, then no data is
82  * written to the buffer and *olen is set to the maximum
83  * length representable as a size_t.
84  *
85  * \note Call this function with dlen = 0 to obtain the
86  * required buffer size in *olen
87  */
